# Titanic Dataset EDA

## Overview
Performed Exploratory Data Analysis on the Titanic dataset using Python.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Female passengers had higher survival rates
- 1st class passengers survived more often
- Children had better survival probability
- Passenger class strongly affected survival
